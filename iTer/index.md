<section lang="en" markdown="1">

iTer is an **IT learning and job interview preparation app** for developers on **iPhone, iPad, and Mac**. Browse **10,000+** offline interview questions across **17 programming languages**, **12 career tracks**, and **16 company question banks — with daily quiz, favorites, wrong-book review, home & lock screen widgets, and native SwiftUI on iPhone, iPad, and Mac.**

## Getting Started

1. Open iTer and browse questions by **language**, **career category**, or **company** on the Categories tab.
2. Tap any question to read the full prompt; tap **Answer** (or press **Space** on keyboard) to reveal the explanation.
3. From a list with multiple questions, use the bottom bar or keyboard to move **Previous / Random / Next**.
4. Try the **Daily Quiz** — 5 multiple-choice questions per day; change the question bank scope from the menu at the top.
5. Swipe a row to **favorite** or **mark mastered**; long-press for learning status.
6. Add **widgets** to Home or Lock Screen; open the app once after install so widgets can sync the offline catalog.
7. Switch display language in **Settings** (English, Simplified Chinese, Traditional Chinese, or Follow System).

## Features

### Question Library

- **10,000+ Questions** — 17 language files, 12 profession tracks, 16 companies (120+ JSON volumes, including 2025 campus interview sets).
- **Fully Offline** — All content bundled; no account or network required.
- **Global Search** — Filter by all / language / profession / company; debounced input, fast indexed lookup.
- **Markdown Answers** — Headings, lists, code blocks, and tables in answers; switch to plain text when needed.
- **Detail Browse Bar** — Previous · Random · **Answer** · Next when opened from a multi-question list; answer scrolls into view on reveal.

### Learning Tools

- **Daily Quiz** — 5 MCQs per day with date seed; optional bank scope (matches search scope & widget).
- **Favorites & Recent** — Save questions; clear all recent history in one tap.
- **Wrong Question Book** — Auto-collected quiz mistakes with review list.
- **Mastery Marks** — Not learned / Needs review / Mastered; swipe or long-press to update.
- **Advanced Learning** — Objc.io & NSHipster bilingual sections, Awesome / Gitbook / practice project links with remark subtitles.

### Personalization

- **15 Built-in Themes** + light / dark / system appearance.
- **UI Languages** — English, Simplified Chinese, Traditional Chinese, Follow System (widgets follow app language).
- **Home Screen Quick Actions** (iOS) — Configure up to **4** shortcuts from **7** options in **Settings → Home Screen shortcuts** (Search, Quiz, Favourites, Wrong Book, Categories, Feedback, Rate). Widgets and **`iTer://`** deep links open questions, quiz, favourites, and wrong book — see [URL Scheme](url-scheme).
- **In-app Safari** for external links.

### iOS & iPad

- **4 Widget Types** — Daily Question, Quiz Glance, Favorites Glance, Wrong Book Glance. Home Screen (Small / Medium where supported) and **Lock Screen** (Inline / Circular / Rectangular on iOS).
- Daily Question widget (iOS 17+): pick question bank scope when adding/editing; tap shuffle to change today's pick.
- Wrong Book widget: deep link to a specific wrong record (`iTer://wrong?record=`).
- **iPad + keyboard**: ⌘F search, ⌘1–⌘4 tabs; detail shortcuts ← / R / Space / → (see below).

### macOS

- Native Mac app with Tab navigation, **Esc** back / dismiss, dedicated **search sheet** (⌘F), and system share panel.
- Default **grid** list layout; independent layout preference from iOS.
- Full keyboard support including detail browse shortcuts.

## Widgets

iTer offers **4 widget kinds** for Home Screen and Lock Screen.

> **Tip:** Open the main app once after install or update so it syncs the question catalog to App Group storage for widgets.

### How to Add

1. Long-press the Home Screen (or Lock Screen) to enter edit mode.
2. Tap **+** and search for **iTer**.
3. Choose a widget and size, then tap **Add Widget**.
4. For **Daily Question** (iOS 17+), long-press the widget → **Edit Widget** to choose question bank scope (All / Language / Profession / Company).

### Available Widgets

| Widget | Description | Sizes |
|--------|-------------|-------|
| Daily Question | One interview question per day; optional scope & shuffle | Small, Medium; Lock Screen |
| Quiz Glance | Today's quiz score, completion, streak | Small; Lock Screen |
| Favorites Glance | Favorite count + preview titles | Small; Lock Screen |
| Wrong Book Glance | Wrong count + summaries; tap to open record (iOS 17+) | Small; Lock Screen |
