### How do I browse interview questions?

Open iTer and use the **Languages** tab for 17 programming language categories, or the **Categories** tab for career tracks, company banks, and advanced learning links. Tap a question to open its detail page. Use **global search** (⌘F on iPad/Mac) to find questions across the entire library with optional scope filters.

### Does iTer work offline?

Yes. All **10,000+** questions and learning resources are bundled with the app. No account, login, or internet connection is required. Favorites, quiz progress, wrong-book records, and preferences are stored locally on your device.

### Why don't my widgets show questions?

After installing or updating iTer, **open the main app once** so it syncs the question catalog to App Group storage. Widgets read from this offline mirror. If widgets still appear empty, launch the app and wait for the library to finish loading, then remove and re-add the widget.

### Can I choose which questions appear in the Daily Question widget?

On **iOS 17+**, long-press the Daily Question widget → **Edit Widget** to pick a scope (All / Language / Profession / Company). You can also tap shuffle on the widget to change today's pick. The in-app daily quiz scope syncs to widgets when you change it in the Quiz tab.

### How do keyboard shortcuts work in question detail?

When you open a question from a list with **2 or more** questions, the bottom bar shows Previous · Random · Answer · Next. On **iPad with an external keyboard** or **Mac**:

| Key | Action |
|-----|--------|
| ← | Previous |
| R | Random |
| Space | Show answer |
| → | Next |

On Mac, **Esc** goes back (or closes share preview first). Shortcuts are disabled at list boundaries or after the answer is already shown.

### What languages does the app UI support?

The interface is available in **English**, **Simplified Chinese**, **Traditional Chinese**, and **Follow System**. Change it in Settings → Display Language. Home screen widgets follow the same language preference.

### What are the Home Screen Quick Actions?

On iPhone and iPad, long-press the iTer icon to use shortcuts configured in **Settings → Home Screen shortcuts** (up to **4** of **7**):

| Type | Action |
|------|--------|
| `iTer://search` | Open global search |
| `iTer://quiz` | Open Daily Quiz |
| `iTer://favorites` | Open Favourites |
| `iTer://wrong` | Open wrong-answer book |
| `iTer://categories` | Open Categories tab |
| `iTer://feedback` | Send feedback (mail) |
| `iTer://review` | Rate the app (StoreKit) |

Widgets and Shortcuts can also use **`iTer://question?id=`**, **`quiz`**, **`favorites`**, and **`wrong`** — see the [URL Scheme reference](url-scheme).

### What platforms are supported?

| Platform | Requirement |
|----------|-------------|
| iPhone / iPad | iOS / iPadOS 16.0+ |
| Mac | macOS 13.0+ |
| Widget scope picker | iOS 17.0+ |

### Where can I download iTer?

iTer is available on the [App Store](https://apps.apple.com/app/id1224934068) for iPhone, iPad, and Mac.

Contact [AppleOSer@gmail.com](mailto:AppleOSer@gmail.com). We typically respond within 48 hours.

### 如何浏览面试题目？

在 **语言** Tab 浏览 17 种编程语言分类，或在 **分类** Tab 进入职业方向、企业真题与进阶学习。点按题目进入详情。**全局搜索**（iPad / Mac 上 ⌘F）可检索全库，并支持按语言 / 职业 / 企业筛选范围。

### 爱技术可以离线使用吗？

可以。全部 **10000+** 题目与学习资源均随 App 打包，无需账号、登录或联网。收藏、测验进度、错题记录与偏好设置均保存在设备本地。

### 小组件为什么没有题目？

安装或更新后，请 **先打开主 App 一次**，以便将题库镜像同步至 App Group，小组件才能离线读取。若仍为空，打开 App 等待题库加载完成，然后移除并重新添加小组件。

### 每日一题小组件可以选择题库范围吗？

**iOS 17+** 可长按「每日一题」小组件 → **编辑小组件**，选择范围（全部 / 语言 / 职业 / 企业）。Widget 上也可 shuffle 换题。App 内每日测验的题库范围变更后会同步至 App Group。

### 题目详情的键盘快捷键怎么用？

从含 **2 道及以上** 题目的列表进入详情时，底部显示 **上一题 · 随机 · 答案 · 下一题**。在 **iPad 外接键盘** 或 **Mac** 上：

| 按键 | 功能 |
|------|------|
| ← | 上一题 |
| R | 随机 |
| 空格 | 显示答案 |
| → | 下一题 |

Mac 上 **Esc** 返回上一级（若正在分享预览则先关闭）。已在首尾题或答案已展开时，对应快捷键无效。

### 应用界面支持哪些语言？

支持 **简体中文**、**繁体中文**、**English** 与 **跟随系统**。在设置 → 显示语言 中切换。主屏小组件会跟随应用语言。

### 主屏快捷指令有哪些？

在 iPhone 或 iPad 上长按爱技术图标，使用 **设置 → 主屏快捷方式** 中配置的快捷项（**7 选 4**）：

| 类型 | 动作 |
|------|------|
| `iTer://search` | 打开全局搜索 |
| `iTer://quiz` | 打开每日测验 |
| `iTer://favorites` | 打开收藏 |
| `iTer://wrong` | 打开错题本 |
| `iTer://categories` | 打开分类 Tab |
| `iTer://feedback` | 发送反馈（邮件） |
| `iTer://review` | 应用评分（StoreKit） |

小组件与快捷指令亦支持 **`iTer://question?id=`** 等深度链接——详见 [URL Scheme 说明](url-scheme)。

### 支持哪些平台与系统版本？

| 平台 | 要求 |
|------|------|
| iPhone / iPad | iOS / iPadOS 16.0+ |
| Mac | macOS 13.0+ |
| 每日一题范围选择 | iOS 17.0+ |

### 在哪里下载爱技术？

可在 [App Store](https://apps.apple.com/app/id1224934068) 下载，支持 iPhone、iPad 与 Mac。

### 仍需帮助？

发邮件至 [AppleOSer@gmail.com](mailto:AppleOSer@gmail.com)。我们通常在 48 小时内回复。
