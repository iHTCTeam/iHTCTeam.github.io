<section lang="en" markdown="1">

iExcel (爱表格) is an **offline Excel learning studio** for **iPhone, iPad, and Mac**. Master **formulas, functions, practical tips, structured lessons, quizzes, and keyboard shortcuts** — all bundled in the app. **No account required.** Track progress with a learning ring and streaks, open **Daily Function**, practice in a **lightweight formula playground**, and stay close with **4 widgets**, **Spotlight**, **Siri / App Shortcuts**, and **`iexcel://` deep links**. Optional **iCloud sync** for favorites, progress, and preferences. **No ads. No tracking.**

> **Note:** Excel is a trademark of Microsoft. iExcel is an independent learning tool and is not affiliated with Microsoft.

## Getting Started

1. Complete the **5-page welcome guide** on first launch (re-open anytime in **Settings → Guides**).
2. On **Home**, check your **learning progress**, **Continue Learning**, and **Daily Function**.
3. Browse **Categories** — popular functions, math, stats, text, lookup, dynamic arrays, tips, lessons, quizzes, and shortcuts.
4. Open any item for rich detail — syntax, parameters, examples, steps, or Mac/Windows key bindings.
5. Mark items as **mastered** when you understand them (opening alone only adds to Recents and your streak).
6. Try the **Formula Playground** with sample cells, or take a **Quiz** and review explanations.
7. Star favorites; enable optional **iCloud Sync** to keep progress across devices.
8. Add **widgets**, use **Home Screen Quick Actions**, or ask **Siri** to open a function or start a quiz.

## Features

### Learning Library

| Module | Count | Highlights |
|--------|------:|------------|
| Functions | 265 | Syntax, parameters, examples, notes, related functions |
| Tips | 71 | Practical spreadsheet workflows with clear steps |
| Lessons | 44 | Structured paths from basics to advanced |

Function categories include Popular, Math & Trig, Statistical, Text, Logical, Lookup & Reference, Date & Time, Financial, Information, Dynamic Arrays, Engineering, and Database.

### Home & Progress

- **Progress ring** — mastered items ÷ learnable total (shortcuts excluded)
- **Continue Learning** — picks up unread recents or the next popular function
- **Daily Function** — one featured function per calendar day
- **Recents** (up to 30) and **Favorites** summaries on Home
- **Streak** — consecutive days you open learning content
- Optional **Favorites** and **Category** tabs (toggle in Settings)

### Formula Playground

A **lightweight practice surface** on fixed sample data (not a full Excel engine):

- Try `SUM`, `AVERAGE`, `MAX`, `MIN`, `IF`, and simple `A1+A2+A3` patterns
- Preset buttons fill examples and evaluate instantly
- Other formulas show a clear “not supported yet” state

Use it to reinforce memory — not as a spreadsheet replacement.

### Search & Detail

- Search by **name, summary, tags, or id** (Chinese & English)
- **iOS 18+** dedicated Search tab; earlier iOS via Quick Actions, deep links, or Intents
- Function pages: copy syntax, jump to related functions
- Tip / lesson pages: body, steps, linked functions
- Quiz flow: answer → reveal → explanation → best score + auto-mastered

### Widgets & System Integration

| Integration | What you get |
|-------------|--------------|

| **Lock Screen** | Daily Function & Progress accessories (iOS) |
| **Home Screen Quick Actions** | Home, Playground, Quiz, Search |
| **Spotlight** | Index of all learning items |
| **Siri / App Shortcuts** | Open Home, Playground, Quiz, or a function by name |
| **`iexcel://` links** | See [Shortcut Links](url-scheme) |

### Privacy

- **No account · no registration · no ads · no tracking**
- All learning content is **bundled offline**
- Optional **iCloud Key-Value Storage** for favorites, progress, quiz scores, streak, and preferences (default off)
- Daily Function id stays **per-device** (not synced)
- Optional tip purchase via **Apple StoreKit** only
