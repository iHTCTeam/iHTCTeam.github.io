<section lang="en" markdown="1">

iRootWord supports **`irootword://`** shortcut links for Shortcuts, Safari bookmarks, widgets, and Home Screen quick actions.

## Link format

```
irootword://{host}?{params}
```

## Content deep links

| URL | Opens |
|-----|-------|
| `irootword://affix?id=42` | Affix detail by ID |
| `irootword://decompose?word=inspection` | Decompose the word "inspection" |
| `irootword://daily` | Today's root |
| `irootword://review` | Start SRS review session |

## Navigation

| URL | Opens |
|-----|-------|
| `irootword://home` | Home |
| `irootword://categories` | Categories browse |
| `irootword://favorites` | Favorites |
| `irootword://search` | Search |
| `irootword://stats` | Learning stats |
| `irootword://settings` | Settings |

## Guides & actions

| URL | Opens |
|-----|-------|
| `irootword://feature-guide` | Feature guide |
| `irootword://widget-guide` | Widget guide |
| `irootword://shortcuts-guide` | Shortcuts guide |
| `irootword://usage-guide` | Usage guide |
| `irootword://feedback` | Feedback email |
| `irootword://in-app-review` | In-app App Store rating |

## Siri & Shortcuts

Built-in App Intents:

- **Root of the Day** — opens today's affix
- **Decompose Word** — decomposes a spoken or typed word
- **Start Review** — begins an SRS session when cards are due
