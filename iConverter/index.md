# iConverter / 爱计算 / 愛計算

**98-in-1 Calculation & Conversion Toolkit for iPhone · iOS 15+**

iConverter is a powerful all-in-one toolkit that brings 98 calculators and converters into a single beautifully designed app. From everyday math to engineering units, from mortgage planning to QR code generation — it's all here, offline, with no ads and no account required.

---

## Getting Started / 快速上手

1. Open iConverter and browse the tool menu by category or use the search bar.
2. Tap any tool to open it. Results update instantly as you type.
3. Tap any result value to copy it to the clipboard (with haptic feedback).
4. Pin your most-used tools to the home screen for one-tap access.
5. Add widgets to your Home Screen or Lock Screen to access tools without opening the app.

## Key Features / 主要特性

- **98 Tools** — Calculators, converters, encoders, date tools, lifestyle helpers, sensors, and more, all in one app.
- **Instant Results** — Fields update in real time as you type. No submit button needed.
- **One-Tap Copy** — Tap any result to copy it, with haptic confirmation and a brief toast notification.
- **Widgets** — Interactive Home Screen and Lock Screen widgets for BMI, BMR, Percentage, Compound Interest, Countdown, World Clock, Lunar Calendar, Date Arithmetic, QR Code, Roman Numeral, and more. Requires iOS 17+ for interactive widgets.
- **URL Scheme** — Open any tool from Safari, Shortcuts, another app, or a QR code using `iconverter://` deep links. Parameters can pre-fill input fields. See the [URL Scheme Reference](url-scheme).
- **Pinned Tools** — Pin up to 8 tools to the home quick-access strip.
- **Home Screen Shortcuts** — Add up to 4 tool shortcuts directly to your iOS Home Screen.
- **Calculator History** — Shared history across all three calculators. Tap any entry to restore it.
- **Unit Copy with Symbol** — Optionally include the unit symbol when copying unit conversion results.
- **Accent Color** — Choose from 12 preset colors or a fully custom color picker.
- **App Icons** — Multiple alternate app icons included.
- **Three Languages** — Simplified Chinese · Traditional Chinese · English, switchable independently of system language.
- **Dark Mode** — Full support for Light, Dark, and System appearance.
- **Offline** — All tools work completely offline. No network connection required (except the Web Browser tool).
- **No Ads · No Account** — Privacy-first. No tracking, no analytics, no sign-up.

## Web Browser Tool / 网页工具

The built-in Web Browser supports:

- Address bar with auto-search (Bing · Google · Baidu · DuckDuckGo · custom engine)
- Bookmarks and navigation history
- Desktop User-Agent toggle
- Auto-refresh timer
- JS debugger injection (Eruda / vConsole / custom script)
- Page source viewer
- In-page text search
- URL QR code sharing
- Cookie import/export
- Keep-awake screen lock override

## Video Frame Export Tool / 视频关键帧

- Pick any video from Photos
- Frame-accurate scrubbing with `AVPlayer` (tolerance = zero)
- Step through frames one by one (long-press for continuous stepping)
- Export current frame to Photos or Share Sheet (JPEG or PNG)
- Optional timecode + frame number watermark
- Batch burst export: configurable radius and step
