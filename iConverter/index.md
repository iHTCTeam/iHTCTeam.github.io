**Everyday math, engineering units, and developer utilities — one beautifully designed app for iPhone and iPad.**

iConverter brings **98 calculators and converters** into a single offline-first toolkit. From mortgage planning to QR scan/generate, from unit conversion to an in-app **Web Browser with NetLog** — tap any result to copy, pin favorites, and jump in from widgets or `iconverter://` deep links.

## Getting Started

1. Open iConverter and browse the tool menu by category or use the search bar.
2. Tap any tool to open it. Results update instantly as you type.
3. Tap any result value to copy it to the clipboard (with haptic feedback).
4. Pin your most-used tools to the home screen for one-tap access.
5. Add widgets to your Home Screen or Lock Screen to access tools without opening the app.

## Key Features

- **98 Tools** — Calculators, converters, encoders, date tools, lifestyle helpers, sensors and more, all in one app.
- **Instant Results** — Fields update in real time as you type. No submit button needed.
- **One-Tap Copy** — Tap any result to copy it, with haptic confirmation and a brief toast notification.
- **QR Code** — Generate codes (multiple formats) or **scan** with the camera; open `iconverter://` links directly.
- **Widgets** — 21 interactive widgets for Home Screen, Lock Screen, and StandBy. **Settings → Widget color style** offers default (per-tool colors), partial accent, or full accent (background included). **About → Widget guide** walks through setup.
- **Launcher widget** — Pick any supported tool for one-tap access (**Web Browser** is excluded — no persistent widget state).
- **URL Scheme** — Open any tool from Safari, Shortcuts, another app, or a scanned QR code using `iconverter://` deep links. Parameters can pre-fill input fields. See the [URL Scheme Reference](url-scheme).
- **Pinned Tools** — Pin up to 8 tools to the home quick-access strip.
- **Home Screen Shortcuts** — Add up to 4 tool shortcuts directly to your iOS Home Screen.
- **Calculator History** — Shared history across all three calculators. Tap any entry to restore it.
- **Unit Copy with Symbol** — Optionally include the unit symbol when copying unit conversion results.
- **Accent Color** — Choose from 12 preset colors or a fully custom color picker.
- **App Icons** — Multiple alternate app icons included.
- **Three Languages** — Simplified Chinese · Traditional Chinese · English, switchable in **Settings → Language** (independent of system language). **Widget configuration UI** (iOS system picker when adding/editing widgets) follows **device system language**, not the in-app language override.
- **Dark Mode** — Full support for Light, Dark and System appearance.
- **Offline-first** — All tools work on-device (except the Web Browser tool, which loads pages from the internet).
- **Privacy-first** — No account, no analytics SDKs, no third-party tracking.

## Web Browser Tool

The built-in **Web Browser** (Encoding category) supports:

- Address bar with auto-search (Bing · Google · Baidu · DuckDuckGo · custom engine) and configurable **home page**
- Bookmarks and navigation history
- Desktop User-Agent toggle
- Auto-refresh timer
- **NetLog** — capture fetch/XHR, console output, and page errors in a filterable sheet (Settings → Web Browser section in app)
- JS debugger injection (Eruda / vConsole / custom script)
- Page source viewer
- In-page text search
- URL QR code sharing
- Cookie import/export
- Keep-awake screen lock override

## Video Frame Export Tool

- Pick any video from Photos
- Frame-accurate scrubbing with `AVPlayer` (tolerance = zero)
- Step through frames one by one (long-press for continuous stepping)
- Export current frame to Photos or Share Sheet (JPEG or PNG)
- Optional timecode + frame number watermark
- Batch burst export: configurable radius and step

- 基於 `AVPlayer` 的逐幀精確定位（容差為零）
- 單幀步進（長按可連續步進）
- 目前幀匯出到相簿或分享面板（JPEG 或 PNG）
- 可選時間碼 + 幀號浮水印
- 批次連拍匯出：可設定半徑與步長

## 小工具

愛計算提供 **21 款互動式小工具**，分為 7 大類，支援主畫面、鎖定畫面和待機模式。

### 新增方法

1. 長按主畫面進入編輯模式。
2. 點按 **+** 按鈕，搜尋「iConverter」。
3. 選擇小工具，滑動選擇尺寸，點按「新增小工具」。
