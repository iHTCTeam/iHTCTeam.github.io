<section lang="en" markdown="1">

iHTCTeam ("we", "us" or "our") built iConverter (also known as "爱计算" / "愛計算"). This page describes our policies regarding data collection, use and disclosure when you use our app.

## Summary

**iConverter does not collect, store or transmit any personal data.** All calculations and conversions happen entirely on your device. No account is required, and no internet connection is needed for any core feature.

## Data Collection

We do not collect any data. Specifically:

- **No personal information** is collected (no name, email, phone number or account).
- **No usage data** or analytics are collected.
- **No third-party SDKs** for advertising, analytics or crash reporting are integrated.
- **No cookies or tracking** technologies are used.
- The Web Browser tool loads web pages from the internet — that traffic goes directly to the websites you visit and is subject to their own privacy policies. iHTCTeam does not intercept or log that traffic.

## Data Stored Locally on Your Device

All data created by iConverter stays on your device:

| Data | Storage | User Control |
|------|---------|--------------|
| Calculator history | Local (UserDefaults) | Delete individual entries or clear all in Settings |
| Bookmarks & browser history | Local (UserDefaults) | Clear in Web Browser settings |
| NetLog entries (Web Browser) | In-memory while session active | Cleared when NetLog is disabled or app terminates |
| Widget accent preference | App Group UserDefaults | Change in Settings → Widget color style |
| App preferences & settings | Local (UserDefaults) | Reset by deleting the app |
| Downloaded files (File Download tool) | Local sandbox / Files app | Delete from the Files app |
| Video frame exports | Photos library | Delete from the Photos app |

Widgets use an **App Group** container to share data between the main app and widget extensions. This container stays on your device and is not accessible to any third party.

## Permissions

iConverter requests the following device permissions, only when a specific feature requires them:

| Permission | Purpose |
|------------|---------|
| Photo Library (Read) | Select videos for the Video Frame Export tool |
| Photo Library (Write) | Save exported video frames |
| Camera | Scan QR codes (optional shortcut) |
| Motion & Fitness | Compass, Level and Altitude sensor tools |
| Location (When In Use) | Altitude tool (barometric altitude requires location for calibration) |
| Local Network | Not requested |
| Contacts / Microphone | Not requested |

Permissions are requested only when you first use the relevant feature and are never used for any purpose other than those listed above.

## Children's Privacy

iConverter does not knowingly collect any personal data from anyone, including children under 13. Since no data is collected at all, no special provisions are required.

## Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page. We encourage you to review this page periodically.

## Contact Us

If you have any questions about this Privacy Policy, please contact us:

- **Email:** [AppleOSer@gmail.com](mailto:AppleOSer@gmail.com)

</section>

**爱计算不收集、存储或传输任何个人数据。** 所有计算与单位换算完全在你的设备上完成。无需账号，所有核心功能也无需联网。

我们不收集任何数据。具体而言：

- **不收集任何个人信息**（无姓名、邮箱、电话或账号）。
- **不收集使用数据**或统计分析。
- **不集成任何第三方 SDK**用于广告、统计或崩溃上报。

## 设备本地存储的数据

爱计算创建的所有数据都留在你的设备上：

| 数据 | 存储位置 | 用户控制 |
|------|---------|--------------|
| 计算器历史 | 本地（UserDefaults）| 在设置中删除单条或清空全部 |
| 书签与浏览历史 | 本地（UserDefaults）| 在网页浏览器设置中清除 |
| NetLog 条目（网页浏览器）| 会话期间内存中 | 关闭 NetLog 或退出 App 后清除 |
| 小组件强调色偏好 | App Group UserDefaults | 在设置 → 小组件颜色风格 中更改 |
| NetLog 条目（网页浏览器）| 会话期间内存中 | 关闭 NetLog 或退出 App 后清除 |
| 小组件强调色偏好 | App Group UserDefaults | 在设置 → 小组件颜色风格 中更改 |
| 应用偏好与设置 | 本地（UserDefaults）| 删除应用即可重置 |
| 下载的文件（文件下载工具）| 本地沙盒 / 「文件」应用 | 在「文件」应用中删除 |
| 视频帧导出 | 相册 | 在「照片」应用中删除 |

小组件使用 **App Group** 容器在主应用与小组件扩展之间共享数据。该容器留在你的设备上，第三方无法访问。

爱计算仅在特定功能需要时请求以下设备权限：

| 权限 | 用途 |
|------------|---------|
| 相册（读取）| 在视频关键帧工具中选择视频 |
| 相册（写入）| 保存导出的视频帧 |
| 相机 | 扫描二维码（可选快捷方式）|
| 运动与健身 | 指南针、水平仪、海拔等传感器工具 |
| 位置（使用应用期间）| 海拔工具（气压海拔需位置以校准）|
| 本地网络 | 不请求 |
| 通讯录 / 麦克风 | 不请求 |

权限仅在你首次使用相关功能时请求，且仅用于上述列明用途。

爱计算不会有意收集任何人（包括 13 岁以下儿童）的个人数据。因为我们完全不收集数据，所以无需特别条款。

我们可能不时更新本隐私政策。任何变更都会发布于本页。建议你定期回访。

- **邮箱：** [AppleOSer@gmail.com](mailto:AppleOSer@gmail.com)
