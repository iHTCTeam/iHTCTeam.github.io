<section lang="en" markdown="1">

iHTCTeam ("we", "us" or "our") built Weber as a **macOS Safari Web Extension** with a minimal host app. This page informs you of our policies regarding the collection, use and disclosure of personal data when you use the extension.

## Summary

**Weber does not collect, store or transmit any personal data.** The extension operates entirely within your browser and does not communicate with any external server.

## Data Collection

We do not collect any data. Specifically:

- **No personal information** is collected (no name, email, phone number or account).
- **No usage data** or analytics are collected.
- **No browsing history or page content** is accessible to us.
- **No cookies or tracking** technologies are used.
- **No third-party services** (advertising, analytics, crash reporting) are integrated.

## How the Extension Works

Weber is a Safari Web Extension that runs locally in your browser. Settings and speed controls live in the **Safari toolbar popup**; the host app only registers the extension. Weber injects a content script into web pages to control HTML5 video playback speed and provides a context menu for copying page or link information as Markdown:

- All processing happens locally within your browser.
- No data is sent to any external server.
- No internet connection is required for the extension to function.

## Permissions

Weber requests the following browser permissions, only as needed:

| Permission | Purpose |
|------------|---------|
| storage | Save your speed preference locally in the browser using `browser.storage.local` |
| tabs | Read the current tab's URL and title for the Markdown copy feature |
| activeTab | Inject the speed-control script into the active tab when you open the popup |
| contextMenus | Add the "Copy as Markdown Format" right-click menu item |
| scripting | Programmatically inject content scripts into pages |
| optional_host_permissions | Only requested if you enable always-autospeed mode; grants access to apply speed on page load |

These permissions are used solely for the stated purposes. No data accessed through these permissions is transmitted externally.

## Data Storage

- **Speed preference** is stored locally in your browser via `browser.storage.local`. This data never leaves your device.
- **No data** is stored on any external server or cloud.
- Uninstalling the extension removes all locally stored data.

## Children's Privacy

Weber does not knowingly collect any personal data from anyone, including children under 13. Since we do not collect any data at all, no special provisions are needed.

## Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated date. We encourage you to review this page periodically.

## Contact Us

If you have any questions about this Privacy Policy, please contact us:

- **Email:** [AppleOSer@gmail.com](mailto:AppleOSer@gmail.com)

</section>
