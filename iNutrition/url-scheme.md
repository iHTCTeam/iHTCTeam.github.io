<section lang="en" markdown="1">

iNutrition supports **`inutrition://`** shortcut links.

## Link format

```
inutrition://{host}/{id}
inutrition://{host}
```

## Content deep links

| URL | Opens |
|-----|-------|
| `inutrition://food/{id}` | Food detail |
| `inutrition://nutrient/{id}` | Nutrient encyclopedia entry |
| `inutrition://article/{id}` | Knowledge article |

## Navigation

| URL | Opens |
|-----|-------|
| `inutrition://home` | Home |
| `inutrition://search` | Search |
| `inutrition://favorites` | Favorites |
| `inutrition://settings` | Settings |
| `inutrition://labelreader` | AI Label Reader (iOS 26+) |

## Tools

| URL | Opens |
|-----|-------|
| `inutrition://tool/bmi` | BMI calculator |
| `inutrition://tool/bmr` | BMR calculator |
| `inutrition://tool/tdee` | TDEE calculator |
| `inutrition://tool` | All tools list |

</section>
