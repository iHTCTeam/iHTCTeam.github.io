<section lang="en" markdown="1">

iSongCi supports **`isongci://`** shortcut links.

## Link format

```
isongci://{host}?{params}
```

## Works & discovery

| URL | Opens |
|-----|-------|
| `isongci://poem?id=42` | Ci detail (poem ID = index in catalog) |
| `isongci://author?id=12` | Author profile |
| `isongci://daily` | Today's Ci |
| `isongci://random` | Serendipity — random curated ci |
| `isongci://continue-reading` | Continue reading |
| `isongci://search?q=明月` | Search for 明月 |

## Navigation

| URL | Opens |
|-----|-------|
| `isongci://home` | Home tab |
| `isongci://category` | Browse tab |
| `isongci://favorites` | Favorites tab |
| `isongci://settings` | Settings tab |

## Guides & feedback

| URL | Opens |
|-----|-------|
| `isongci://usage-guide` | Usage guide |
| `isongci://widget-guide` | Widget guide |
| `isongci://shortcuts-guide` | Shortcuts & Siri guide |
| `isongci://home-shortcuts` | Home Screen quick actions |
| `isongci://feature-guide` | Replay welcome guide |
| `isongci://feedback` | Feedback email |
| `isongci://in-app-review` | In-app review |

## Siri & Shortcuts

Three App Intents ship with the app:

| Intent | Example phrase |
|--------|----------------|
| Open Today's Ci | "Today's ci in iSongCi" |
| Random Ci | "Serendipity in iSongCi" |
| Search Ci | "Search in iSongCi" (with keyword) |

Use in **Shortcuts** automations — e.g. open `isongci://daily` every morning after your alarm.
