<section lang="en" markdown="1">

Tang Poems 300 supports **`itangpoems://`** shortcut links.

## Link format

```
itangpoems://{host}?{params}
```

## Poems & authors

| URL | Opens |
|-----|-------|
| `itangpoems://poem?id=1` | Poem detail (e.g. id 1) |
| `itangpoems://author?id=1` | Poet detail |
| `itangpoems://recite?id=1` | Recitation mode for a poem |

## Discovery

| URL | Opens |
|-----|-------|
| `itangpoems://daily` | Daily poem |
| `itangpoems://random` | Random poem |
| `itangpoems://favorites` | Favorites |
| `itangpoems://search?q=月` | Search for poems containing 月 |

</section>
