<section lang="en" markdown="1">

iConnect is a peer-to-peer communication app for Apple devices on the same local Wi-Fi network. **Chat one-to-one or in groups, push-to-talk (PTT), voice messages, and transfer files** — photos, videos, documents and any file type — directly between iPhone, iPad and Mac. **No cloud, no account, no internet required.**

## Getting Started

1. Open iConnect on two or more devices connected to the same Wi-Fi network.
2. Nearby devices appear automatically on the radar screen.
3. Tap a device to connect, or join a group chat from the Chats tab.
4. Send text / photos / voice messages, hold **PTT** to talk, or open the **Transfers** tab for files and clipboard text.
5. The receiving device confirms file transfers (unless the sender is a trusted device).
6. Completed transfers appear in the **Transfers** tab and transfer history.

See the in-app **Usage Guide** (Settings) for a full walkthrough.

## Features

### Communication

- **Instant Discovery** — Nearby devices appear on an animated radar screen.
- **Chat & Group Chat** — Text messaging with reply, **recall** (within 2 minutes), **@mentions**, link previews, and **in-conversation search**.
- **Voice Messages** — Record and send voice bubbles; playback speed up to **2×** from the bubble menu.
- **Push-to-Talk (PTT)** — Hold-to-talk over the local P2P link; no phone number or server. On Mac, **Esc** closes the PTT sheet.
- **Forward & Multi-select** — Forward messages to other chats; batch actions from conversation settings.
- **In-App Safari** — http(s) links open inside the app (iOS Safari View / macOS sheet) instead of jumping away.
- **Delivery / Read Receipts** — Message state stays local between peers.

### File Transfer

- **Transfers Tab** — Dedicated send/receive UI with real-time progress.
- **Any File Type** — Photos, videos, PDFs, documents, audio, archives (ZIP, RAR, 7z), and more.
- **Configurable File Size** — Presets or custom; choose **unlimited** if you need no cap.
- **Batch Transfer** — Select and send multiple files at once.
- **Clipboard Sharing** — Send copied text directly to another device.
- **Trust Devices** — Auto-accept from trusted peers.
- **Transfer History** — Quick Look preview, save, and share.

### General

- **Cross-Device** — iPhone, iPad, and Mac in any combination.
- **Privacy First** — Chat, voice and files stay on your local network. Nothing is sent to the cloud.
- **9 App Icons** — Default + 8 alternate Home Screen icons (iOS).
- **URL Scheme** — Open tabs and actions via **`iconnect://`**. Long-press the icon for Quick Actions on iOS. See the [URL Scheme Reference](url-scheme).
- **Tip Jar** — Optional in-app tip to support development.
- **Dark Mode** — Light / Dark / System.
- **3 UI Languages** — English, Simplified Chinese, Traditional Chinese (independent of system language).

- **信任设备** — 信任对端自动接收。
- **传输历史** — Quick Look 预览、保存与分享。

### 其他
