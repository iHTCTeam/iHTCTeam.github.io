<section lang="en" markdown="1">

iConnect is a peer-to-peer communication app for Apple devices on the same local Wi-Fi network. **Chat one-to-one or in groups, push-to-talk (PTT), and transfer files** — photos, videos, documents and any file type — directly between iPhone, iPad and Mac. **No cloud, no account, no internet required.**

## Getting Started

1. Open iConnect on two or more devices connected to the same Wi-Fi network.
2. Nearby devices appear automatically on the radar screen.
3. Tap a device to connect, or join a group chat from the Chats tab.
4. Send messages, hold the PTT button to talk, or open the Transfers tab to send files, photos, or clipboard text.
5. The receiving device confirms file transfers (unless the sender is a trusted device).
6. Completed transfers appear in the **Transfers** tab and transfer history.

See the in-app **Usage Guide** (Settings) for a full walkthrough of chat, PTT, transfers and trust devices.

## Features

### Communication

- **Instant Discovery** — Nearby devices appear automatically on an animated radar screen.
- **Chat** — Text messaging with connected peers on your local network.
- **Group Chat** — Create or join multi-device group conversations.
- **Push-to-Talk (PTT)** — Hold-to-talk voice over the local P2P link; no phone number or server required.

### File Transfer

- **Transfers Tab** — Dedicated tab for sending and receiving files with real-time progress.
- **Any File Type** — Photos, videos, PDFs, documents, audio, archives (ZIP, RAR, 7z), and more.
- **Configurable File Size** — Set a per-file size limit in Settings (presets or custom); choose **unlimited** if you need no cap.
- **Batch Transfer** — Select and send multiple files at once.
- **Clipboard Sharing** — Send copied text directly to another device.
- **Real-Time Progress** — Live transfer speed and estimated time remaining.
- **Trust Devices** — Mark devices as trusted for automatic acceptance.
- **Transfer History** — Log with Quick Look preview, save, and share.

### General

- **Cross-Device** — Works between iPhone, iPad, and Mac in any combination.
- **Privacy First** — All chat, voice and file data stay on your local network. Nothing is sent to the cloud.
- **8 Alternate App Icons** — Pick a Home Screen icon that fits your style.
- **URL Scheme** — Open tabs and in-app actions via **`iconnect://`** from Safari, Shortcuts, or other apps. Long-press the icon for Quick Actions on iOS. See the [URL Scheme Reference](url-scheme).
- **Usage Guide** — Built-in guide in Settings covering discovery, chat, PTT, transfers and trust.
- **Tip Jar** — Optional in-app tip to support development.
- **Dark Mode** — Supports light and dark appearance.
- **3 UI Languages** — English, Simplified Chinese (简体中文), and Traditional Chinese (繁體中文). Switch in Settings; independent of system language.

## 快速上手

1. 在两台或多台连接到同一 Wi-Fi 网络的设备上打开「爱联」。

### 通讯

- **即刻发现** — 附近设备自动出现在动画雷达界面。

- **对讲（PTT）** — 按住说话，通过本地 P2P 链路传语音；无需电话号码或服务器。

### 文件传输

- **传输标签页** — 专用标签页发送与接收文件，显示实时进度。
- **支持任意类型** — 照片、视频、PDF、文档、音频、压缩包（ZIP、RAR、7z）等。
- **可配置文件大小** — 在设置中设定单文件大小上限（预设或自定义）；可选 **不限**。
- **批量传输** — 一次选择并发送多个文件。
- **剪贴板分享** — 把复制的文本直接发送到其他设备。
- **实时进度** — 显示实时传输速度与剩余时间。
- **可信任设备** — 标记设备为可信任，自动接收文件。
- **传输历史** — 历史记录，支持 Quick Look 预览、保存和分享。

### 其他

- **全平台互通** — iPhone、iPad、Mac 任意组合互传。
- **隐私优先** — 聊天、语音与文件数据均只在本地网络传输，不上云。
- **8 款备用图标** — 在设置中更换主屏幕应用图标。
- **URL Scheme** — 通过 **`iconnect://`** 从 Safari、快捷指令或其他 App 打开标签页与应用内操作；iOS 上长按图标可使用 Quick Actions。详见 [URL Scheme 文档](url-scheme)。
- **使用指南** — 设置内置指南，涵盖发现、聊天、对讲、传输与信任设备。
- **打赏** — 可选应用内打赏支持开发。
- **深色模式** — 完整支持浅色与深色外观。
- **3 种界面语言** — 英文、简体中文、繁体中文。在设置中切换，与系统语言无关。
