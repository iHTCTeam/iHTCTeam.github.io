<section lang="en" markdown="1">

iYuanQu supports **`iyuanqu://`** shortcut links.

## Link format

```
iyuanqu://{host}?{params}
```

## Works & discovery

| URL | Opens |
|-----|-------|
| `iyuanqu://work?id=1` | Qu detail (e.g. id 1) |
| `iyuanqu://daily` | Daily qu |
| `iyuanqu://last-read` | Last read qu |
| `iyuanqu://search?q=秋` | Search for works containing 秋 |

## Navigation

| URL | Opens |
|-----|-------|
| `iyuanqu://home` | Home |
| `iyuanqu://categories` | Categories |
| `iyuanqu://categories?section=gongdiao` | Categories, jump to a section |
| `iyuanqu://favorites` | Favorites |
| `iyuanqu://settings` | Settings |
| `iyuanqu://reading-history` | Reading history |

## Guides & feedback

| URL | Opens |
|-----|-------|
| `iyuanqu://usage-guide` | Usage guide |
| `iyuanqu://widget-guide` | Widget guide |
| `iyuanqu://shortcuts-guide` | Shortcuts guide |
| `iyuanqu://feature-guide` | Feature guide |
| `iyuanqu://feedback` | Feedback email |
| `iyuanqu://in-app-review` | In-app review |
