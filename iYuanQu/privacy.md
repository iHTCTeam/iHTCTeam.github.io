iHTCTeam built iYuanQu (爱元曲 / 愛元曲).

## Summary

**Data Not Collected.** iYuanQu makes **zero outbound network requests** after install. **No account. No ads. No analytics SDK.** All qu data is stored locally. Optional **iCloud Sync** for favorites, playlists, reading history and font size uses **your private iCloud** only.

## Data Stored on Your Device

| Data | Purpose |
|------|---------|
| Qu database | 8,967 bundled qu works |
| Favorites & playlists | Works you star and organize |
| Reading history | Recently viewed works |
| Preferences | Theme, accent color, reading layout, font size |
| Notification settings | Daily qu reminders (local only) |

## iCloud Sync

Optional sync for favorites, playlists, reading history and font size through your private iCloud key-value store.

## Permissions

| Permission | When | Why |
|------------|------|-----|
| Notifications | Daily qu reminder | Local scheduling only — no server |
| iCloud | Optional sync | Sync favorites, playlists, history and font size |

## Contact

[AppleOSer@gmail.com](mailto:AppleOSer@gmail.com)

<section lang="zh-CN" markdown="1">

iHTCTeam 开发了爱元曲。

## 摘要

**不收集数据。** 安装后**零外发网络请求**。**无账号、无广告、无分析 SDK。** 所有曲作数据保存在本地。可选 **iCloud 同步**收藏、曲单、浏览记录与字号，仅使用**你的私有 iCloud**。

## 设备本地存储

| 数据 | 用途 |
|------|------|
| 曲作数据库 | 内置 8,967 首曲作 |
| 收藏与曲单 | 你加星标并整理的曲作 |
| 浏览记录 | 最近查看的曲作 |
| 偏好设置 | 主题、主题色、阅读布局、字号 |
| 通知设置 | 每日一曲提醒（仅本地调度） |

## iCloud 同步

可选通过私有 iCloud 键值存储同步收藏、曲单、浏览记录与字号。

## 权限

| 权限 | 时机 | 原因 |
|------|------|------|
| 通知 | 每日一曲提醒 | 仅本地调度——无服务器 |
| iCloud | 可选同步 | 同步收藏、曲单、浏览记录与字号 |

## 联系

[AppleOSer@gmail.com](mailto:AppleOSer@gmail.com)
