<section lang="en" markdown="1">

iDictionary supports **`idict://`** shortcut links for Shortcuts, Safari bookmarks, widgets, and Home Screen quick actions.

## Link format

```
idict://{host}?{params}
```

## Lookup

| URL | Opens |
|-----|-------|
| `idict://char?c=爱` | Character detail for 爱 |
| `idict://word?id=123` | Word entry by ID |
| `idict://idiom?id=456` | Idiom entry by ID |
| `idict://search` | Search screen |

## Daily & Random

| URL | Opens |
|-----|-------|
| `idict://daily` | Daily character card |
| `idict://random` | Random character |

## Input modes

| URL | Opens |
|-----|-------|
| `idict://handwriting` | Handwriting lookup |
| `idict://ocr` | Camera OCR scan |

## Navigation

| URL | Opens |
|-----|-------|
| `idict://home` | Home |
| `idict://categories` | Browse categories |
| `idict://favorites` | Favorites |
| `idict://settings` | Settings |

## Guides & actions

| URL | Opens |
|-----|-------|
| `idict://feature-guide` | Feature guide |
| `idict://widget-guide` | Widget guide |
| `idict://shortcuts-guide` | Shortcuts guide |
| `idict://usage-guide` | Usage guide |
| `idict://feedback` | Feedback email |
| `idict://in-app-review` | In-app App Store rating |

</section>
