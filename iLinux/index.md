<section lang="en" markdown="1">

iLinux is a privacy-first, fully offline **shell command reference and learning app** for **Linux**, **Windows CMD/Batch**, and **PowerShell** — on **iPhone, iPad and Mac**. Browse **1650+** commands per content language, read Markdown documentation, compare cross-platform equivalents, take daily quizzes scoped to each shell library, and track favourites, learning status and wrong answers. Core libraries work **without any account**; optional **iCloud sync** keeps progress aligned across your Apple devices.

> **Important:** iLinux is a local reference and learning companion. It helps you look up and study shell commands on your devices. **It does not connect to or execute commands on a real terminal, remote server or any other machine.**

## Getting Started

1. Open iLinux. Use the **Shell platform** menu in the toolbar (terminal icon) to choose **Linux**, **CMD/Batch** or **PowerShell**.
2. Browse commands from **Commands**, or switch to **Categories** for topic-based exploration. On **Mac**, pick a section from the **sidebar** instead of bottom tabs.
3. Open any command to read its summary, full Markdown documentation, and **cross-platform equivalents** (when available).
4. Go to **Daily Quiz** to practise with today's quiz set for the **current shell platform**.
5. Press **⌘F** (or tap the magnifier on iOS) to search by command name or description.
6. Star a command to add it to **Favourites**; set **learning status** (Unlearned / Learned / Mastered) from the detail page or list actions.
7. Open **Settings** to customise theme, language (iOS/iPad), list/grid layout, favourites scope, **iCloud sync**, widget shell library (iOS/iPad), and Home Screen shortcuts (iOS/iPad).

## Features

### Cross-Platform Equivalents

**100+ equivalent groups** link Linux, CMD and PowerShell commands — for example `ls` ↔ `dir` ↔ `Get-ChildItem`. The **Equivalents** bar on command detail pages lets you jump to another platform in one tap or click.

### Commands

- **A–Z Index** — Optional **Favourites** and **Recent** sections (toggle in Settings).
- **List & Grid Views** — **1–6 columns on iPhone**, **1–8 on iPad and Mac** (each device remembers independently).
- **Quick Filters** — All, Favourites, Recent, Unlearned.
- **Full-Text Search** — Within the current shell library; results update as you type.
- **Command Detail** — Summary + Markdown with syntax highlighting.
- **Learning Status** — Unlearned / Learned / Mastered.
- **Share** — Export a **Markdown file** or **styled PNG share card**.

### Daily Quiz

- Daily quiz set **scoped to the active shell platform**.
- **Match Summary** and **Match Option Meaning** question types.
- **Random Refresh** and **Perfect Streak** (per platform).
- Wrong answers saved to the **Wrong Answer Book**.

### Learning Progress & iCloud Sync

- **Favourites** with **scope** (all platforms vs current platform only).
- **Recent**, **learning status**, and **wrong-answer records**.
- **iCloud Sync** (optional, Settings → Learning stats): when signed into the same Apple ID, syncs **favourites**, **quiz progress**, **theme**, and **display preferences** (language, shell platform, favourites scope, list/grid) across **iPhone, iPad and Mac**. Conflicts merge automatically (e.g. favourites union). Disable per device without affecting others. Requires iCloud; command libraries remain fully offline.

### macOS Desktop Experience

Native **macOS 13.0+** app (not Mac Catalyst):

- **Sidebar shell** — Commands, Categories, Daily Quiz and Settings in a `NavigationSplitView` layout.
- **Window sizing** — minimum **900×560**, default **1100×720**; resizable desktop window.
- **Extended keyboard shortcuts** — **⌘,** Settings; **⌘[** / **Esc** back; **⌘← / ⌘→** previous/next category; quiz keys **1–4**, **⌘R** reshuffle, **Return** next question.
- **System Settings-style toggles** and desktop-optimised sheets.
- **Share** via the macOS share picker; **feedback** via `mailto:` or your mail client.

**Mac limitations (by design):** Home Screen **widgets**, **Home Screen quick actions**, and **alternate app icons** are **iOS/iPad only**. Settings shows a notice — configure widgets on iPhone or iPad.

### iPhone & iPad

- **Tab bar** navigation (Commands / Categories / Quiz / Settings).
- **5 Home Screen & Lock Screen widgets** with configurable shell library and Daily Command shuffle (iOS 17+).
- **Home Screen quick actions** — up to 4 from 7 options.
- **6 app icons** — default plus 5 alternates.
- **Haptic feedback** on key actions.

### URL Scheme

`ilinux://` deep links work on **iOS, iPad and Mac** (command detail with optional `platform=`, quiz, favourites, wrong book). Home Screen quick-action-only URLs (`search`, `categories`, `feedback`, `review`) remain **iOS/iPad only**. See [URL Scheme Reference](url-scheme).

### Themes & Language

- **15 colour presets** + **custom accent**; Light / Dark / System.
- **UI & content language:** English, 简体中文, 繁體中文, or System — picker in **Settings → Language** on **iOS/iPad**; preference **syncs to Mac via iCloud** when enabled.
