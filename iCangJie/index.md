<section lang="en" markdown="1">

**iCangJie** is a comprehensive **Cangjie input method** learning and character lookup tool for **iPhone, iPad, and Mac**. With a curated library of **27,000+ Han characters** covering Cangjie 3rd and 5th Gen standards, radical breakdown, full/quick/mixed **typing practice**, a card-style **daily quiz**, 11 HTML tutorials, pinyin & Jyutping search, converter tools, **8 widgets**, and optional **iCloud sync**, it is the all-in-one companion for mastering Cangjie — **offline-first, privacy-respecting, zero third-party SDKs.**

Built with pure SwiftUI.

## Why iCangJie?

Most dictionary apps stop at cold codes. iCangJie is built for **learners who actually type**:

- See **why** a code is built that way — radical highlights and keyboard strips on every detail page.
- **Drill** with three typing modes, 16 classical passages (Simplified & Traditional), and wrong-book retry.
- **Quiz yourself** with a **configurable daily quiz** (5–15 questions, custom word source & code length).
- **Track progress** with 7/30-day typing reports and JSON backup for your learning data.
- **Glance** at progress on the Home Screen, Lock Screen, or StandBy with 8 purpose-built widgets.
- Ask **Siri** to look up a Cangjie code or start today's quiz; index favorites in **Spotlight**.
- Switch the entire app to **English, Simplified Chinese, or Traditional Chinese** — UI refreshes instantly.

## Getting Started

1. Complete the **5-screen onboarding** on first launch (or skip anytime). Follow the **Newcomer Path** card in **Learn** for quick links.
2. Open **Lookup** and search by Cangjie code, character, pinyin prefix, or Jyutping — or tap the **search prompt card** on the resting screen.
3. Browse **Levels** by code length (1–5) or quick-method buckets; shuffle for random practice.
4. Enter **Learn** for HTML tutorials and pinch-zoom radical charts (adjustable reading size).
5. Try **Typing** — pick full-code, quick-code, or mixed mode; open **Practice Report** for trends.
6. Take the **Daily Quiz** — customize questions in quiz settings. Review mistakes in the Wrong Book.
7. Add **widgets** to your Home Screen or Lock Screen. Configure up to **4 Home Screen Quick Actions** and optional **Spotlight indexing** in Settings.

## Features

### Dictionary & Lookup

- **27,000+ curated characters** — Cangjie 3rd & 5th Gen; offline bundled JSON cleaned for renderable Han characters only.
- **Four char-set priorities** — Hong Kong, Taiwan Traditional, Simplified, and General — affect search ranking.
- **Four search modes** — Cangjie code prefix, exact character, pinyin prefix, and Jyutping prefix; smart hints when a mode returns no results.
- **Home screen cards** — Favorites, recent views, and quick code-length shortcuts (toggleable).
- **Word detail** — Large character tile, colored code badges, per-code radical keyboard strips; **3rd/5th Gen side-by-side compare** for single characters; **one-tap copy** for character or codes; learning status, favorites, **Youdao online lookup**, and **long-image share**.

### Levels (Code Length)

- Browse the full table by **3rd / 5th Gen** × **code length 1–5** or **quick-method** buckets.
- **List or grid** layout with device-aware column counts; pull to **shuffle**.

### Typing Practice

- **Three modes** — Full code, quick code (levels 1–2), and mixed (try quick first, fall back to full after a mistake).
- **Content sources** — 16 built-in classical passages (SC/TC JSON), random word lists, pasted text, favorites, and wrong-book retry (auto-resolves the correct Han character).
- Live **radical key highlights**; optional passage pinyin; virtual keyboard with wrong-key **keep or clear** preference.
- Session stats (accuracy, streak) and **daily aggregates** (up to 120 days) sync via iCloud when enabled.
- **Practice Report** — 7/30-day typing trend chart and code-length distribution; open from Typing hub or Settings → Learning Stats.

### Daily Quiz & Wrong Book

- **Configurable quiz** — 5 / 8 / 10 / 15 questions per day; word source: dictionary, favorites, or wrong book; code-length filter 1–5; toggle question types.
- Three types: character→code, code→character (with keyboard strip), and character→first radical key.
- Card-style UI with instant feedback and end-of-round **wrong-answer review**. **Reshuffle** for a fresh set. Gear icon opens **Quiz Settings**.

### Learn (Tutorials & Onboarding)

- **5-screen onboarding** on first launch (re-run anytime from **Settings → App Settings**); **Newcomer Path** card in Learn tab (lookup, quiz, first tutorial, widget guide).
- **11 HTML lessons** + grouped **radical chart images** in the Learn hub.
- **Adjustable reading size** (85%–160%); HTML adapts to light/dark; Traditional script when app language is zh-Hant.
- Pinch-zoom charts; **swipe gallery** between images at 1× zoom; **save to Photos** or share (iOS).

### Tools (Settings)

- **Pinyin & Cangjie converter** — Multiple output styles (toned, plain, numbered, annotated, aligned grid); optional Cangjie codes; copy or **share preview image**.
- **Text-to-speech playground** — Play, pause, and stop.
- **Widget guide** — Full catalog with Lock Screen steps and FAQ (iOS).
- **Home Screen Quick Actions** — Pick up to 4 from 7 options (search, quiz, favorites, wrong, levels, feedback, rate).
- **Spotlight indexing** (iOS) — Surface favorites & recents in system search (toggle in Settings).
- **Siri & Shortcuts** — Built-in App Intents: look up Cangjie code, start daily quiz, open lookup.
- **JSON backup & restore** — Export/import learning data (merge or replace) from Settings → Learning Stats.

### Personalization

- **15 preset gradient themes** + custom accent color; light / dark / system appearance.
- **Tile grid follows theme** — Character tiles in list/grid views use your accent color.
- **In-app language** — English, Simplified Chinese, Traditional Chinese, or follow system; widgets mirror language via App Group.
- List/grid layout preferences; pinyin overlay toggle; default Cangjie version and char-set.

### iOS & iPad

- **8 widget kinds** — Daily Character, Code Length, Daily Quiz, Typing Glance, Favorites, Wrong Book, Recent, Learning Progress. Home Screen, Lock Screen, and StandBy.
- Daily Character **shuffle button** on small and medium widgets (iOS 17+). Code Length widget: pick **1–5** when adding; **shuffle button** on small and medium to refresh the trio (iOS 17+).
- Favorites / Wrong / Recent widgets show **2×3 (small) / 3×3 (medium)** rice-grid previews with codes.

### macOS

- Native Mac app with transparent navigation chrome, **Esc** back navigation, system share sheet, and full typing practice.
- Lookup search integration; floating Safari sheet for external links in tutorials.

### Privacy

- **Offline core** — Dictionary, quizzes, tutorials, and typing work without network.
- **No ads · No tracking** — No analytics SDKs. Optional iCloud uses Apple KVS only.
- Photo library access is requested **only when you choose to save a radical chart image** (add-only).

- 字根圖捏合縮放、1× 時畫廊左右滑、**儲存相簿/分享**（iOS）。

### 工具（設定）

- **拼音/倉頡轉換器** — 多格式輸出 + 可選倉頡碼；複製或 **預覽長圖分享**。
- **語音朗讀** — 播放 / 暫停 / 停止。
- **小工具教學** — 新增步驟、鎖屏說明、8 款小工具圖鑑與 FAQ。
- **主畫面快速操作** — 7 項中選 4 項。
- **Spotlight 索引** — 收藏與最近瀏覽寫入系統搜尋（iOS，可開關）。
- **Siri / 捷徑** — 查倉頡碼、開始測驗、開啟查字。
- **JSON 備份與恢復** — 設定 → 學習統計；合併或覆蓋匯入。

### 個人化

- **15 套主題預設** + 自訂取色；淺色/深色/跟隨系統。
- **米字格跟隨主題** — 列表/網格中的漢字格使用你的主題色。
- **應用內三語**；小工具經 App Group 同步語言與主題色。
- 列表/網格、拼音顯示、預設倉頡版本與字集等偏好。

### iOS 與 iPad

- **8 款小工具** — 每日一字、碼長精選、每日測驗、打字一覽、收藏、錯題、最近、學習進度；主畫面、鎖屏、StandBy。
- 每日一字小/中尺寸 **換字**；碼長小工具新增時選 **1–5 碼**，小/中尺寸 **換一組**（iOS 17+）。
- 收藏/錯題/最近為中/小尺寸 **米字格網格** 預覽。

### macOS

- 原生 Mac 體驗：**Esc** 返回、系統分享、完整打字練習；教程外鏈系統瀏覽器開啟。

### 隱私

- **核心離線** — 字典、測驗、教程、打字無需連網。
- **無廣告 · 無追蹤** — 無統計 SDK；iCloud 僅用 Apple KVS。
- 僅在 **儲存字根圖到相簿** 時請求相簿寫入權限。
