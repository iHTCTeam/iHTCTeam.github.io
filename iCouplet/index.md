iCouplet (爱对联 / 愛對聯) is an **immersive couplet learning studio** — not just another couplet list. **No account required.** All **3,369 curated couplets** (~102 authors, ~3,348 famous lines) are bundled on your device: **zero network requests** after install. **Recite** with karaoke-style highlighting; enjoy **vertical reading** (品联); practice in the **Study classroom** (fade, fill-blank, recitation check, quiz, and 对对子); and open a **contextual Daily Couplet** tied to solar terms, festivals, and seasons. Export **12-style share cards**, add **home / lock screen widgets**, optional **Reading Live Activity**, **Siri & Shortcuts**, and optional **iCloud** for favorites and progress. **No ads. No tracking. Data Not Collected.**

## Getting Started (step by step)

### 1. First launch

1. Install from the [App Store](https://apps.apple.com/app/id6793778879) and open **iCouplet**.
2. Complete the **6-page welcome guide** (brand → recite → vertical reading → study → system features → privacy).
3. You can **replay the feature guide** anytime: **Settings → Help → Feature Guide**.
4. In-app **Usage / Widget / Shortcuts** guides live under **Settings → Help** and cross-link each other.

### 2. Read today’s couplet on Home

1. Open the **Home** tab.
2. The **Daily Couplet** hero shows the title plus **upper and lower lines** (and a horizontal scroll title / 横批 when present).
3. Tap the hero to open the full detail page.
4. Optional: turn on **Daily Couplet notifications** in **Settings → Daily Couplet** (local scheduling only).

### 3. Browse & search

1. Open **Browse** (or Home search on older iOS builds).
2. Explore five sections: **form (体裁)**, **theme**, **author (联家)**, **famous lines**, and **collections**.
3. Use **Search** for titles, lines, and authors (Simplified / Traditional friendly).
4. Tap any couplet for detail; star it to **Favorites**.

### 4. Recite (诵联)

1. On a couplet detail page, tap **Recite**.
2. Follow **karaoke-style** highlighting (paragraph-level in detail; word-level in the Recitation studio).
3. Adjust **rate / pitch / voice** in **Settings → Reading & Recitation**.
4. Optional: enable **play in silent mode**, **background audio** (Now Playing / lock screen), and continuous playlists.
5. Deep link / Siri **recite** can open a couplet and start speaking automatically.

### 5. Vertical reading & share cards (品联 / 联卡)

1. On detail, switch **vertical / horizontal** layout; toggle **pinyin**, traditional characters, and centering as you like.
2. Open **Share Card** → pick among **12 styles** (classic / ink / vertical / cinnabar / …) and aspect ratios.
3. Preview, **save to Photos** (add-only permission), or share via the system sheet.
4. Optional: enable **Reading Live Activity** in Settings (iPhone; must start in foreground).

### 6. Study classroom (对课学堂)

1. Open **Study** from Home / deep link `icouplet://study` / Shortcuts.
2. Try modes: **Fade**, **Fill-blank**, **Recitation check**, **Quiz**.
3. Play **对对子** (pair matching game).
4. Track **4-level mastery** (none → familiar → known → mastered) and streaks.

### 7. Widgets, Siri & Home Screen shortcuts

1. Long-press Home Screen → **Edit Widgets** → add **Daily Couplet / Famous Line / Study Progress / Favorites**.
2. On Daily Couplet widgets, try **paper styles** (soft / solid red / plain) and **desktop refresh** to change today’s couplet (also syncs today’s notification text).
3. Ask **Siri** for today’s couplet, recite, random, favorites, or search.
4. Configure up to **4 Home Screen quick actions** in **Settings → Shortcuts**.

### 8. Privacy & sync

1. Everything works **fully offline** after install.
2. Optional **iCloud Sync** (Settings → Data) mirrors favorites, couplet lists, reading history, mastery, and streaks through **your private iCloud** — not iHTCTeam servers.
3. Tip the developer (optional StoreKit purchase) never unlocks paywalled content and never creates an account.

## Features

### Recitation Studio

- Karaoke-style **TTS** with highlighting
- Adjustable **speed**, **pitch**, and **voice**
- Optional **background audio** + lock-screen **Now Playing**
- **Continuous reading** queues from favorites / categories

### Vertical Immersive Reading

- Classical **vertical & horizontal** layouts
- **Pinyin**, traditional conversion, background / appreciation notes
- Line order always **横批 (if any) → upper → lower**
- **12 share-card styles** with multiple aspect ratios

### Study System

- Fade · fill-blank · recitation check · quiz
- **对对子** game mode
- **4-level mastery** + streaks

### Contextual Daily Couplet

- Solar terms · festivals · seasons (with same-day cache / desktop refresh)
- Local notifications (~60-day schedule window)
- Widgets + Control Center controls (iOS 18+)

### Content at a Glance

- **3,369 couplets** · **~102 authors** · **~3,348 famous lines** · ~3.9 MB bundled data
- Five browse sections; list / grid / card previews
- **100% offline** — no network, no account, no ads

### Widgets, Siri & System Integration

- Daily / famous / progress / favorites widgets (+ lock screen)
- Reading **Live Activity** (optional)
- Spotlight indexing; **`icouplet://` deep links** — see [Shortcut Links](url-scheme)
- Settings guides: Usage · Widget · Shortcuts · Feature Guide replay

### Privacy

- **Zero outbound network requests** after install (except optional Apple StoreKit for tipping, and system share targets you choose)
- **No account · no ads · no analytics SDK**
- Local App Group data + optional private iCloud KVS

## 保姆级上手教程

### 1. 第一次打开

1. 从 [App Store](https://apps.apple.com/app/id6793778879) 安装并打开**爱对联**。
2. 跟随 **6 页欢迎引导**（品牌欢迎 → 诵联 → 竖排品鉴 → 对课学堂 → 系统能力 → 隐私离线）。
3. 之后可随时：**设置 → 帮助 → 功能引导** 重播（不会改写「已完成引导」状态）。
4. **设置 → 帮助** 内还有互相跳转的**使用说明 / 小组件指南 / 快捷指南**，建议按需打开对照操作。

### 2. 在首页看「每日一联」

1. 打开**首页** Tab。
2. Hero 卡片会显示题名 + **上联与下联**（有横批时按「横批 → 上联 → 下联」展示）。
3. 点按 Hero 进入详情，可读赏析、开诵联、做对课、分享联卡。
4. 想每天提醒：到 **设置 → 每日一联** 打开通知并设定时间（**仅本机调度**，无服务器）。

### 3. 分类浏览与搜索

1. 打开**分类** Tab（若关闭了分类 Tab，可从首页入口或深链进入）。
2. 五分区：**体裁 / 主题 / 联家 / 名句 / 合集**。
3. 用**搜索**查题名、联句、联家（简繁均可）。
4. 点进任意一联收藏为星标，之后在**收藏** Tab 管理联单。

### 4. 诵联（怎么听、怎么连读）

1. 在联详情点**诵联**。
2. 详情页为段落高亮；**诵联工作台**可字级高亮，更适合跟读。
3. 到 **设置 → 阅读与诵联** 调节语速、音调、音色。
4. 建议按需打开：
   - **静音模式仍外放**（默认开）
   - **后台诵联**（锁屏 Now Playing；远程「下一首」仅播放列表有效）
   - **连续播放**
5. 也可用 Siri / 深链 `icouplet://recite` 或 `icouplet://recite?id=` 直接开读。

### 5. 竖排品鉴与联卡分享

1. 详情页切换竖排/横排，按喜好开拼音、繁体、居中。
2. 打开**联卡** → 选 **12** 种风格与比例（方形 / Story 等）。
3. 预览后可**存入相册**（仅「添加照片」权限）或系统分享。
4. iPhone 可在设置中打开**品联实时活动**（须在前台启动；离开详情会结束）。

### 6. 对课学堂（怎么练到掌握）

1. 从首页或 `icouplet://study` 进入对课。
2. 四种模式：**渐隐 / 填空 / 对课检测 / 测验**；另有**对对子**。
3. 行序与展示一致：有横批时横批为第 0 行。
4. 掌握度四级：未学习 → 已了解 → 已学会 → 已掌握；配合打卡连续天数。

### 7. 小组件、Siri、主屏快捷（一步步加到桌面）

1. 长按主屏幕空白处 → **编辑** → **添加小组件** → 搜索「爱对联」。
2. 可选：**每日一联 / 名句 / 学习进度 / 收藏**（含锁屏尺寸）。
3. 每日一联小组件可改**联纸样式**（淡彩 / 朱纸 / 素面），并用桌面「刷新」换联（会同步今日通知正文与落地 ID）。
4. 对 Siri 说今日一联、随机、诵联、收藏或搜索。
5. **设置 → 快捷**：最多勾选 **4** 个主屏长按快捷（默认含每日 / 随机 / 诵联 / 收藏）。

### 8. 数据、同步与隐私（放心用）

1. 对联库装在 App 内，**日常使用不需要网络**。
2. **设置 → 数据 → iCloud 同步**（可选）：同步收藏、联单、浏览记录、掌握度与打卡——走**你的私有 iCloud**，不经过我们的服务器。
3. 「打赏开发者」为可选内购，**不解锁功能、不创建账号**。
4. 完整说明见 [隐私政策](privacy)。
